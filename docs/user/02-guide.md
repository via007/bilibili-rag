# 使用指南

> 完整功能说明

---

## 功能概览

| 功能 | 说明 |
|------|------|
| 登录 | B 站扫码登录 |
| 收藏夹 | 同步、管理 |
| 知识库 | 入库、ASR |
| 对话 | 问答、检索 |
| 导出 | Markdown |

---

## 登录

1. 点击「登录」
2. B 站 APP 扫码
3. 确认授权

---

## 收藏夹

- 刷新：同步最新
- 展开：查看视频
- 状态：未入库/已入库/有更新

---

## 知识库

1. 勾选收藏夹
2. 点击「入库」
3. 等待处理完成

---

## 对话

1. 输入问题
2. 发送
3. 查看回答和来源

---

## 导出

### 视频/收藏夹

1. 选择内容
2. 点击导出
3. 选择格式
4. 下载

### 会话总结

1. 打开会话
2. 点击导出总结
3. AI 提取知识点
4. 下载
