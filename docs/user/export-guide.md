## 功能介绍

导出功能可以将知识库中的内容导出为 Markdown 格式，方便你：

- 📥 保存到本地
- 📝 整理笔记
- 📤 分享给他人
- 🔍 离线查阅

---

## 导出视频

### 步骤

1. 打开收藏夹页面
2. 展开收藏夹，找到目标视频
3. 点击视频右侧的导出按钮
4. 选择导出格式（完整/精简）
5. 点击下载

### 格式说明

| 格式 | 内容 |
|------|------|
| 完整格式 | 标题 + 摘要 + 内容提纲 + 原文内容 |
| 精简格式 | 标题 + 摘要 + 核心要点 |

---

## 导出收藏夹

### 步骤

1. 在收藏夹列表，勾选要导出的收藏夹
2. 点击「导出」按钮
3. 选择导出格式
4. 点击下载

### 支持多选

- 可以同时导出多个收藏夹
- 每个收藏夹内容用 `---` 分隔

## 导出会话

### 步骤

1. 打开对话页面
2. 点击「导出」按钮
3. 选择「导出会话」（问答格式）
4. 下载 Markdown 文件

### 导出内容

```markdown
# 会话标题

## 对话记录

### 用户
问题内容

---

### AI回答
回答内容
*来源: BVxxxx*

---
```

## 导出会话总结

> 🤖 AI 自动提取知识点，生成结构化总结

### 什么是会话总结？

不同于问答格式的导出，会话总结会：

- 从 AI 回答中**提取核心知识点**
- 按**主题自动归类**
- 带上**相关视频来源**

### 示例

```markdown
# Python 学习总结

> 来源: 2 个视频 · 8 轮对话

## 整体总结
介绍了 Python 核心数据结构的特点与使用场景...

## 核心知识点

### 列表（List）
- 有序可变的数据结构
- 常用方法：append/remove/pop

### 元组（Tuple）
- 有序不可变的集合

## 相关视频
- [Python 入门教程](https://...)
```

### 使用步骤

1. 打开有对话的会话
2. 点击「导出总结」按钮
3. 预览总结内容
4. 点击下载

### 缓存机制

- ✅ **首次导出**：调用 AI 生成总结，自动缓存
- ✅ **再次导出**：直接返回缓存，秒级响应
- 🔄 **重新生成**：点击「重新生成」可强制刷新

### 缓存状态

导出弹窗会显示：

- 📊 来源视频数、对话轮次
- 🔢 版本号（v1, v2, ...）
- ⏰ 生成时间

## 常见问题

### Q: 导出失败怎么办？

A: 请检查：
- 网络是否正常
- 是否已登录
- 目标内容是否存在

### Q: 会话总结需要付费吗？

A: 首次生成会消耗 Token，之后使用缓存免费。

### Q: 如何更新已缓存的总结？

A: 点击「重新生成」按钮，确认后会自动更新。

### Q: 导出的文件在哪里查看？

A: 浏览器下载文件夹，或者弹窗中点击「复制」粘贴到笔记软件。

### Q: 支持导出为其他格式吗？

A: 目前仅支持 Markdown 格式。
